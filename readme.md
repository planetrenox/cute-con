# Usage

This library provides a set of utilities for logging:

## 1. The `_` Function

- **Description**: Acts like `console.log` but returns the last argument passed. This behavior is unconventional but can be useful in certain contexts.
- **Usage**:
  ```javascript
  import { _ } from 'cute-con';

  const result = _('This will be logged and this will be returned');
